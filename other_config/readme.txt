== completion.bashとkey-bindings.bashについて ==
fzfのキーバインドとかの設定が記載されているファイル、
linuxbrewでfzfをインストールしたのでこのファイルは自分でダウンロードしてきて
それを.for_others_from_bash_profileから読み込むといった運用にする。
(本体が更新されたら動かなくなるかもしれないから都度githubから持ってくる)


.gitconfig.userはリモートにアップしていない自分で下記のようにgithubユーザ名とメールアドレスを記載する

[user]
	name = <username>
	email = <email>
